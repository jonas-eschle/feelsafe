# TRUTH, read by tool/coverage/filter_coverage.sh).

# Each non-comment line is ONE `lcov --remove` glob, matched against the lcov
# `SF:` source paths (repo-relative). The GA coverage target is "~99% of the
# LOGIC surface" (owner M5 mandate + CLAUDE.md D6), so the denominator must be
# human-authored, host-testable logic.
#
# HONESTY CONTRACT (non-negotiable): a glob may ONLY exclude code that has NO
# host-testable logic — generated code, or a platform passthrough whose only
# host-reachable statements are a platform-guard early-return + a MethodChannel/
# static-plugin call that throws MissingPluginException off-device. A file that
# contains parsing / state / fallback / formatting decisions stays in the
# denominator EVEN IF it is currently 0% covered — that is a C6/C7 coverage gap
# to CLOSE, never something to hide behind an exclusion.
#
# Measured against `flutter test --coverage` on 2026-06-09 (4068 host tests):
# raw total = 26705 instrumented lines, of which 50.2% (13396) are generated.
